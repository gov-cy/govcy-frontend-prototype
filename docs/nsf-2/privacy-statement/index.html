Πολιτική απορρήτου – οι προσωπικές σας πληροφορίες

Αυτή η πολιτική απορρήτου εξηγεί πώς SERVICE_OWNER χρησιμοποιεί πληροφορίες σχετικά με εσάς, για να διεκπεραιώσει την αίτησή σας για έκδοση εγγράφου.

Τύποι πληροφοριών που συλλέγουμε και χρησιμοποιούμε

Συλλέγουμε τις ακόλουθες πληροφορίες απευθείας από εσάς:

  • διεύθυνση επικοινωνίας
  • διεύθυνση email
  • αριθμός τηλεφώνου

Συλλέγουμε τις ακόλουθες πληροφορίες για εσάς από το CY Login:

  • όνομα χρήστη στο Cy Login
  • διεύθυνση email
  • αριθμός ταυτότητας ή ARC

Χρησιμοποιούμε τις ακόλουθες πληροφορίες για εσάς από τα δικά μας αρχεία:

  • ονοματεπώνυμο
  • ημερομηνία γέννησης, φύλο, εθνικότητα και κοινότητα στην οποία ανήκετε
  • διεύθυνση επικοινωνίας και email (αν ισχύει)
  • αριθμός τηλεφώνου (αν ισχύει)
  • ημερομηνία θανάτου (αν ισχύει)
  • ονοματεπώνυμο, πατρικό επώνυμο, υπηκοότητα, τόπος και ημερομηνία γέννησης, φύλο, σχέση με αιτητή και ημερομηνία θανάτου για: παιδιά, γονείς, παππούδες, γιαγιάδες και εάν ο αιτητής είναι άρρενας, εγγόνια (αν ισχύει).

Αν αφήσετε σχόλια στο Βοηθήστε μας να βελτιώσουμε την υπηρεσία, συλλέγουμε επίσης αυτές τις πληροφορίες:

  • ο βαθμός ικανοποίησής σας με τη χρήση της υπηρεσίας
  • η γνώμη και οι εισηγήσεις σας για το πώς μπορούμε να βελτιώσουμε την υπηρεσία.

Γιατί χρησιμοποιούμε τις πληροφορίες σας

Συλλέγουμε και χρησιμοποιούμε τις πληροφορίες αυτές για την έκδοση του εγγράφου που επιλέξατε. Στην περίπτωση αίτησης, οι εξεταστές μας τις χρησιμοποιούν για να επεξεργαστούν την αίτησή σας.

Χωρίς πρόσβαση στις πληροφορίες αυτές, δεν μπορούμε να εκδώόσουμε το έγγραφο που επιλέξατε ή να επεξεργαστούμε την αίτησή σας.

Αν αφήσετε σχόλια στο Βοηθήστε μας να βελτιώσουμε την υπηρεσία, θα τα χρησιμοποιήσουμε για να αξιολογήσουμε πώς αισθάνονται οι χρήστες για την εμπειρία τους από τη χρήση της υπηρεσίας. Αυτό γίνεται για να βρούμε και να προτεραιοποιήσουμε τομείς βελτίωσης και να πάρουμε ιδέες για νέες εξελίξεις.

Πώς αποθηκεύονται οι προσωπικές σας πληροφορίες

Οι πληροφορίες σας αποθηκεύονται με ασφάλεια σε διακομιστές με έδρα την Κύπρο.

Τα δικαιώματά σας

Επικοινωνήστε στο XXXXX@papd.mof.gov.cy εάν θέλετε:

  • να λάβετε αντίγραφο των πληροφοριών που διατηρούνται για εσάς
  • να περιορίσετε την επεξεργασία των δεδομένων σας
  • να διορθώσετε ή να ενημερώσετε τις προσωπικές σας πληροφορίες
  • να υποβάλετε ένσταση για την επεξεργασία των προσωπικών σας πληροφοριών
  • να μεταφερθούν τα στοιχεία σας σε άλλο οργανισμό ή σε εσάς.
